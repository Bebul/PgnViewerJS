parameter pieceStyle

By using the parameter pieceStyle, you will get different figures for the choosen theme or the default theme. Each theme may define its own default figure set. So look at the directory ../themes to see those.
  • The first setup is with explicit pieceStyle "wikipedia".
  • The second is with the implicit pieceStyle "wikipedia" (as default value for the parameter)
  • The third setup is with explicit pieceStyle "alpha".
  • The fourth setup is with explicit pieceStyle "uscf".
  • The fifth setup is with explicit pieceStyle "case".
  • The sixth setup is with explicit pieceStyle "condal".
  • The seventh setup is with explicit pieceStyle "maya".
  • The eighth setup is with explicit pieceStyle "merida".
  • The nineth setup is with explicit pieceStyle "leipzig".
  • The tenth setup is with explicit pieceStyle "chesscom".
  • The eleventh setup is with explicit pieceStyle "beyer".
