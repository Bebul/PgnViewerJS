parameter coordsInner

The parameter "coordsInner" denotes if the coordinates should be shown in the board and not outside. Those are the numbers 1-8 for the rows and the alpha characters a-h for the columns on the board.
  • The first setup is with explicit coordsInner "true".
  • The next is with the implicit coordsInner "true" (as default value for the parameter)
  • The last one is with the explicit coordsInner "false" to move the coordinates outside the board.
