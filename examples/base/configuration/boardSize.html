Parameter boardSize

The following boards (and moves) set the parameter boardSize:
  • Implicit by using the size of the div Uses the default of Chessground, not the div.
  • Implicit by using the parameter size. Uses the parameter width all the time.
  • Explicit to set the board width
  • Explicit to set the board width big (just to show the difference)
  • Explicit to set the board width small (for pgnPrint usage).
  • Explicit to set the board width medium (for the viewer or editor).
