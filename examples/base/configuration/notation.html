parameter showNotation

The parameter "showNotation" denotes if the notation should be shown. Those are the numbers 1-8 for the rows and the alpha characters a-h for the columns on the board.
  • The first setup is with explicit showNotation "true".
  • The next is with the implicit showNotation "true" (as default value for the parameter)
  • The last one is with the explicit showNotation "false" to hide the notation.
