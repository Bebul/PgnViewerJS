Here some comments what a reasonable structure can be:

  • Moves have to be captured in one div, with a special class.
  • There should be an option to style moves per move number:
    • Only break before white is moving.
    • Break all the time before white is moving.
    • Do that only for main lines, not variations.
  • Comments have to be part of the move:
    • Before comment directly before the move (even for white before or after the move number??)
    • After comment directly after the move
  • Decide if variations would be included in parentheses or some other thing. This can be done easily directly in the CSS (only).
