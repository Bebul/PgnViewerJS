Set coordsFontSize in configuration

The following is shown: default (coordsFontSize == 11px); set coordsFontSize == 24px. Useful in all mode view, edit, board, not for print. And only useful if showCoords == true. Can be combined with coordsInner == true/false. When resizing the board (by the user), the coordsFontSize will not be adjusted and remain the same as before. You cannot use any other measure that is allowed in CSS (em, ...) besides px, px is taken all the time.
