Set pieceStyle in CSS

Shows how to set the pieceStyle in CSS. First: pieceStyle used from outer HTML element (look at body tag). Second: pieceStyle == merida (which does not work, see #345 that shows it. Third: pieceStyle == wikipedia.
The strange thing here is that the order of the pieceStyles in the CSS file decide which one is shown, or can be overwritten. I have made some more examples to show the effect.

Here visible (from left to right):

  • wikipedia (default from the outside)
  • wikipedia (because later in CSS than merida) ==> not working
  • alpha (because later in CSS than wikipedia)
